# LifeCare-Pharmacy

A fully responsive WordPress and PHP-based website designed for performance, scalability, and seamless user experience.

## Thesis Project Overview
This repository contains the complete WordPress export data (`.xml` file) for the LifeCare Pharmacy website. It includes all posts, pages, custom post types, products, and configurations required for the thesis evaluation.

## How to Import & Review
1. Setup a local WordPress environment (using XAMPP or LocalWP).
2. Go to the WordPress Dashboard -> **Tools** -> **Import**.
3. Install and run the **WordPress Importer**.
4. Upload the provided `.xml` file to restore the complete project data and content.
